# working
Automotive electronic rearview mirrors

该项目用于替代汽车的光学后视镜，系统包括布置在车外的摄像头、布置在车内的 LCD 屏幕和电子控制
单元。驾驶员可以通过 LCD 屏幕观察到侧后方的路况。在雨天、雾天、夜晚等路况下，电子控制单元针对图像进行
相应处理以提供良好的视野。
